# ShuttlePoseReview v1.2.0

电脑端 1.2 主要同步 Android 端已经稳定下来的正式复盘体验，继续以 2D 动作复盘为核心，围绕重发力时间线、慢动作回看、证据卡和历史记录做补齐。

## 更新重点

- **重发力慢动作复盘**：2D 复盘页支持围绕重发力窗口自动慢放，便于查看引拍、发力和收拍过程。
- **慢动作速度选择**：支持关闭慢动作，也支持不同慢放速度，避免快节奏对抗视频全程都被放慢。
- **重发力统计收敛**：当前主统计回到明显重发力窗口，避免把轻挡、随挥和遮挡抖动计入击球次数。
- **可读证据卡**：击球时机、发力链、回位恢复会以更容易理解的档位、解释、置信度和训练建议呈现。
- **历史复盘入口**：Web 端主页最多保留最近 20 条本机复盘记录，可直接打开旧视频结果。
- **综合质量参考**：不再强调外部等级标签，改为“综合质量参考”和 `S / A / B / C / D` 档位，避免把单次动作复盘误读成真实水平评定。

## 修复内容

- 修复重发力窗口慢放逻辑过宽，导致快节奏对抗视频大段时间都处于慢动作的问题。
- 修复部分复盘记录刷新后不方便继续打开旧结果的问题。
- 优化证据卡文案，不再直接暴露难理解的底层数值。

## 使用提示

电脑端仍建议先使用 15-60 秒的训练或对抗片段。当前版本更适合观察明显挥拍动作，例如杀球、高远球、平抽和重发力过渡；放网、轻挡等小动作可能不会单独计入重发力窗口。

## 说明

ShuttlePoseReview 当前不会直接判断真实球速、真实发力大小或真实竞技等级。系统基于人体 2D 骨架计算关节角、速度、相对高度、躯干尺度和动作窗口，再映射为复盘指标。结果适合个人训练复盘和动作讨论，不建议作为比赛水平或教学定级依据。
