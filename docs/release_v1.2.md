# ShuttlePoseReview Android 1.2

ShuttlePoseReview 1.2 主要补齐 Android 端的正式复盘体验：从相册选择视频、端上完成分析，到主页面查看重发力、慢动作、证据卡和历史记录。

## 更新重点

- **相册选择视频**：Android 端优先打开系统相册/视频选择器，不再优先进入文件管理器；不支持时会自动回退到系统文件选择器。
- **60 秒视频处理**：导入视频后默认处理前 60 秒，更适合连续回合、训练片段和多拍对抗复盘。
- **复盘主页面**：处理完成后进入正式复盘页，集中展示骨架叠加视频、重发力状态、三项分数和时间线。
- **重发力慢动作**：播放经过重发力窗口时自动慢放，也可以点击时间线跳到某一次重发力附近。
- **慢动作可调节**：支持 `0.75x / 0.50x / 0.33x` 三档速度，并可关闭慢动作。
- **骨架更顺滑**：骨架覆盖层加入前后姿态帧插值，低速回看时减少跳帧感。
- **历史复盘**：主页最多保留最近 20 条本机复盘记录，旧视频结果可以直接翻阅。
- **可读证据卡**：击球时机、发力链、回位恢复都可以展开查看，证据以档位、解释、置信度和训练建议呈现。
- **弱化等级表达**：不再强调外部等级标签，改为“综合质量参考”和 `S / A / B / C / D` 档位，避免把单次动作复盘误读成真实水平评定。

## 修复内容

- 修复 Android 端时间线多次点击后，视频人物和骨架覆盖层容易不同步的问题。
- 修复直接播放视频时，重发力窗口不会自动进入慢动作的问题。
- 修复 Android 端慢动作没有开关和速度调节的问题。
- 优化 0.25x 低速播放卡顿体验，改为更稳定的 0.33x 档位。

## 使用方式

1. 安装 APK。
2. 打开应用，点击“输入视频并开始复盘”。
3. 从系统相册中选择羽毛球视频。
4. 等待端上分析完成。
5. 在复盘主页面查看重发力时间线、慢动作回看、三项分数和证据卡。

## 适用视频

- 建议使用 15-60 秒训练或对抗片段。
- 近端球员需要尽量清晰入镜，身体和持拍手不要长期被遮挡。
- 当前更适合杀球、高远球、平抽、重发力过渡等明显挥拍动作。
- 放网、轻挡等小动作可能不会单独计入重发力窗口。

## 说明

ShuttlePoseReview 当前不会直接判断真实球速、真实发力大小或真实竞技等级。系统基于人体 2D 骨架计算关节角、速度、相对高度、躯干尺度和动作窗口，再映射为复盘指标。结果适合个人训练复盘和动作讨论，不建议作为比赛水平或教学定级依据。
